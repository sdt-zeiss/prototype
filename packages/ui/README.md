# UI

This package contains the UI components for the application.

## Usage

package.json

```json
{
  "dependencies": {
    "s3": "workspace:*"
  }
}
```

```jsx
return (
  <Card>
    <CardContent>
      <CardHeader>
        <CardTitle>{title}</CardTitle>
      </CardHeader>
    </CardContent>
  </Card>
);
```
